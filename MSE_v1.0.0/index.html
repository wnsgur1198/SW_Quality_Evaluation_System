MSE Project

정의

본 프로젝트는 업체에서 개발한 의료용 소프트웨어가 의료국제표준을 통과하는지 검사하는 프로그램을 개발하는 프로젝트이다. 의료용 소프트웨어가 국제 표준에 적합하여 상용화가 가능한지 검사하는 것을 개발 목적으로 하며, 소프트웨어를 평가하고 인증하는 인증기관(TTA)을 대상으로 한다.

목적

본 프로젝트는 업체에서 개발한 의료용 소프트웨어가 의료국제표준을 통과하는지 검사하여 인증 조건을 만족하는지 검사하고, 검사결과를 분석 자료로서 제공해 어느 항목이 부적절한지 확인할 수 있어 체계적으로 소프트웨어를 관리할 수 있는 것을 목적으로 한다.
